# Security Policy

## Supported Versions
PathX is currently in research and simulation phase. Security updates are provided for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v1.2.0  | :white_check_mark: |
| < v1.2.0| :x:                |

## Reporting a Vulnerability
As this is a safety-critical research project, we take security seriously. Please do not report security vulnerabilities through public GitHub issues. 

Instead, please contact the maintainer directly through GitHub profile contact options. We aim to respond within 48 hours.
